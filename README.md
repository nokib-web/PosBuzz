# POSBuzz - Point of Sale System

POSBuzz is a modern, comprehensive Point of Sale (POS) system designed for efficiency and scalability.

## Project Structure
- `frontend/`: React-based user interface.
- `backend/`: Node.js/Express server logic and API.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
To run both frontend and backend in development mode:
```bash
npm run dev
```

### Production
To build and run for production:
```bash
npm run build
npm start
```
